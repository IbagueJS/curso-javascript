# do while

La sentencia _do while_ a diferencia de _while_ se ejecuta una vez antes de que se evalue la expresión condicional

Un bucle _do while_ se parece a lo siguiente:

```js
var contador = 0;
do {
  console.log(contador++);
} while (contador < 5);
```
