# Metodo has

El m√©todo `has()` devuelve un valor booleano (verdader o false, true o false) que indica si un elemento con la clave especificada existe o no.

```js
  const map = new Map()
  map.set('nombre', 'julian')

  console.log(map.has('nombre'))
```
